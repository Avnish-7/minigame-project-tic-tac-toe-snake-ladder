🎮 Tic-Tac-Toe (C++)

A classic, console-based 2-player Tic-Tac-Toe game developed in C++. The game uses a 3x3 grid (2D array) and is structured using Object-Oriented Programming (OOP) principles for a clean and replayable experience.

🚀 Features

👥 2-Player Local Multiplayer ('X' vs 'O')
🔢 Intuitive 1-9 numpad-style input mapping
🛡️ Robust input validation (prevents overwriting spaces or out-of-bounds moves)
🏆 Automatic Win and Draw detection
🔄 "Play Again" feature with automatic board reset
📑 Clean, continuously updating console interface

🛠️ Technologies Used

C++ (Object-Oriented Programming)

📂 Project Structure

tic-tac-toe-cpp/
│
├── main.cpp             # Main source code containing the TicTacToe class
├── README.md            # Project documentation

▶️ How to Run

Download or clone the repository
Open the project folder
Compile the code using a C++ compiler (e.g., `g++ main.cpp -o tictactoe`)
Run the executable file (`./tictactoe` or `tictactoe.exe`)

🧠 Concepts Used

Object-Oriented Programming (Classes, Methods, Encapsulation)
2D Arrays (`char board[3][3]`)
Nested Loops (for board generation and win checking)
Mathematical Coordinate Mapping (converting 1D input to 2D grid locations)
Object Lifecycle & Scope (for cleanly resetting the game state)

📌 Note

The grid inputs map exactly to the numbers 1 through 9. Simply type the number corresponding to the square you want to claim and hit Enter.
